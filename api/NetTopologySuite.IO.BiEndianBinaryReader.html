    Class BiEndianBinaryReader

    Extends the class to allow reading values in the specified format.

    Inheritance
    Object
    BiEndianBinaryReader
    Namespace: NetTopologySuite.IO
    Assembly: NetTopologySuite.dll
    Syntax
    public class BiEndianBinaryReader : BinaryReader
    Remarks

    While BiEndianBinaryReader extends adding methods for reading integer values (ReadInt32()) and double values (ReadDouble()) in the specified format, this implementation overrides methods, such and and more, for reading values in the specified by Endianess format.
